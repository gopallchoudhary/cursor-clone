## üé® Fonts Used

The project uses a combination of custom and Google Fonts:

- **Cursor Gothic** (Custom)
  - Primary brand font
  - Used for headings and main UI text

- **Inter**
  - Fallback UI font
  - Ensures readability across devices

- **Poppins**
  - Used selectively for supporting text and smooth typography

- **Outfit**
  - Modern UI font for balance and spacing

```css
font-family: "CursorGothic", "Inter", sans-serif;
```
